CSS

  • Separate CSS file created to define styling for a page, and is then linked using
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • The border property assigns a border with variable width in px
  • The box-shadow adds shadow effect around elements frame
  • box-shadow desrcibed by x and y offset relative to the element, blur and spread radius, and color
  • HTML hexadecimal code defines the color
  • Multiple effects are separated by commas

  • for loops used to run code from loop start to loop end defined by var x = 0; x < 'variablename'.length; x++
